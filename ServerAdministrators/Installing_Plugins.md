#  Installing Plugins

From BukkitWiki

Jump to: navigation, search

This guide explains how to install most basic plugins on your server. You must
already have a running CraftBukkit server set up and have knowledge of how to
use the Minecraft server console. If you do not have a server set up please go
to [Setting up a server](/Setting_up_a_server) and follow the instructions
there. This guide does not cover setting up an SQL database or editing
properties files, please consult the forums to see if you need this. Always
read the plugin instructions if there are any.

  

## Contents

  * 1 Installing Most Plugins
    * 1.1 Notes
  * 2 Updating Plugins
    * 2.1 Notes

## [[edit](/index.php?title=Installing_Plugins&action=edit&section=1)]
Installing Most Plugins

  1. Download a plugin of your choice. 
  2. Place the .jar and any other files in your plugins directory. 
  3. Run the server and wait for it to fully load. 
  4. Type **stop** in your Minecraft server console to bring the server to a clean stop. 
  5. Run the server. 
  6. All done! Your plugin should be installed and ready to be used. 

#### [[edit](/index.php?title=Installing_Plugins&action=edit&section=2)] Notes

    ![Lightbulb.png](http://hydra-media.cursecdn.com/wiki.bukkit.org/6/6f/Lightbulb.png?version=0e5a9741aa6598c745ae089a294d0510) **Note**: You can find plugins at the [Bukkit Plugin List](http://plugins.bukkit.org/) or at [BukkitDev](http://dev.bukkit.org/categories/)
    ![Lightbulb.png](http://hydra-media.cursecdn.com/wiki.bukkit.org/6/6f/Lightbulb.png?version=0e5a9741aa6598c745ae089a294d0510) **Note**: Make sure to download the right version for the plugin 
    ![Lightbulb.png](http://hydra-media.cursecdn.com/wiki.bukkit.org/6/6f/Lightbulb.png?version=0e5a9741aa6598c745ae089a294d0510) **Note**: The file you downloaded may be an archive (.zip, .rar, .7z) and will need to be extracted using an archive manager such as [7zip](http://www.7-zip.org). 
    ![Lightbulb.png](http://hydra-media.cursecdn.com/wiki.bukkit.org/6/6f/Lightbulb.png?version=0e5a9741aa6598c745ae089a294d0510) **Note**: Your plugins directory will be a folder called "plugins" in the folder where you created your CraftBukkit server. 
    ![Lightbulb.png](http://hydra-media.cursecdn.com/wiki.bukkit.org/6/6f/Lightbulb.png?version=0e5a9741aa6598c745ae089a294d0510) **Note**: After you have started your server it may have generated a configuration file, check with the plugin's forum or wiki page for any properties you can configure. 
    ![Lightbulb.png](http://hydra-media.cursecdn.com/wiki.bukkit.org/6/6f/Lightbulb.png?version=0e5a9741aa6598c745ae089a294d0510) **Note**: Make sure to check your log and console for errors created from your newly installed plugin, this may indicate it requires additional setting up (MySQL, permissions) check with the forum post or the BukkitDev page to see if this is the case. 

## [[edit](/index.php?title=Installing_Plugins&action=edit&section=3)]
Updating Plugins

  1. Create a directory called "update" in your plugins directory. 
  2. Download the plugin that you wish to update 
  3. Place ONLY the .JAR file into the update directory. 
  4. Restart your CraftBukkit server. 
  5. All done! Your plugin should have been updated. 

#### [[edit](/index.php?title=Installing_Plugins&action=edit&section=4)] Notes

    ![Lightbulb.png](http://hydra-media.cursecdn.com/wiki.bukkit.org/6/6f/Lightbulb.png?version=0e5a9741aa6598c745ae089a294d0510) **Note**: Ensure the name of the .jar in the update directory is the same as the name of the .jar in the plugins directory for the same plugin. 
    ![Lightbulb.png](http://hydra-media.cursecdn.com/wiki.bukkit.org/6/6f/Lightbulb.png?version=0e5a9741aa6598c745ae089a294d0510) **Note**: The **reload** command will also unload all the plugins, copy the files from the update directory into the plugins directory (replacing any files with the same name) then load all the plugins again. 
    ![Lightbulb.png](http://hydra-media.cursecdn.com/wiki.bukkit.org/6/6f/Lightbulb.png?version=0e5a9741aa6598c745ae089a294d0510) **Note**: This method is safer than replacing the file yourself while the server is running and will not disconnect all your players if it was to be done manually.
